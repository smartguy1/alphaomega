How to auto detect the genome build from bam?

Auto-detecting the genome build from a BAM is not 100% guaranteed, but in practice you can get very high confidence by combining a few signals.

Here’s a robust, production-grade approach (what real pipelines do).

🧬 1) Check the BAM header (first and easiest step)

Run:

samtools view -H input.bam

Look for:

🔹 Sequence dictionary (@SQ lines)
@SQ SN:chr1 LN:248956422

The chromosome lengths are the key.

🔍 Compare chromosome lengths
Chromosome	GRCh37	GRCh38
chr1	249,250,621	248,956,422

👉 If you see:

248,956,422 → GRCh38
249,250,621 → GRCh37

This alone usually solves it.

🔹 Look for explicit tags (sometimes present)

You might see:

@PG ID:bwa VN:0.7.17 CL:bwa mem -t 8 GRCh38.fa ...

or:

@CO reference=GRCh37

👉 If present, this is definitive.

🧠 2) Chromosome naming convention (quick heuristic)
Style	Likely build
chr1, chr2, chrX	UCSC style (often GRCh37 or GRCh38)
1, 2, X	Ensembl style

👉 Not definitive, but useful signal.

🔬 3) Probe known SNP positions (high-confidence method)

This is the most reliable fallback.

Idea:

Check reads at positions where builds differ.

Example SNP:

rsID position differs between builds
Steps:
samtools mpileup -r chr1:752566 input.bam

Then test:

GRCh37 position vs GRCh38 position

👉 Whichever shows real coverage = correct build

🧪 4) Use reference fingerprinting (best practice in pipelines)

Compare BAM header against known references:

Extract sequence lengths:
samtools view -H input.bam | grep @SQ | awk '{print $2,$3}'

Then compare against:

Known GRCh37 dictionary
Known GRCh38 dictionary

👉 Exact match = definitive answer

🤖 5) Automate it (Python example)

Using pysam:

else:
    print("Unknown build")

Here’s a production-ready automated genome build detection module you can drop into your pipeline. It combines:

Header-based detection (fast path)
Sequence dictionary fingerprinting (robust)
Optional SNP probing fallback (deterministic)
🧬 Overview

It distinguishes between:

GRCh37
GRCh38

and returns a confidence score + reason.

🐍 Python module (drop-in)

This uses pysam.

import pysam

GRCH37_CHR1 = 249250621
GRCH38_CHR1 = 248956422

def detect_build_from_header(bam_path):
    bam = pysam.AlignmentFile(bam_path, "rb")

    refs = dict(zip(bam.references, bam.lengths))

    # Try both naming conventions
    chr1_len = refs.get("chr1") or refs.get("1")

    if chr1_len == GRCH37_CHR1:
        return {
            "build": "GRCh37",
            "confidence": 0.95,
            "method": "chr1_length"
        }

    if chr1_len == GRCH38_CHR1:
        return {
            "build": "GRCh38",
            "confidence": 0.95,
            "method": "chr1_length"
        }

    return {
        "build": "unknown",
        "confidence": 0.0,
        "method": "chr1_length_failed"
    }


def detect_build_from_full_dictionary(bam_path):
    bam = pysam.AlignmentFile(bam_path, "rb")

    refs = dict(zip(bam.references, bam.lengths))

    # Known signatures (subset)
    GRCH37_SIG = {
        "chr1": 249250621,
        "chr2": 243199373,
        "chr10": 135534747,
    }

    GRCH38_SIG = {
        "chr1": 248956422,
        "chr2": 242193529,
        "chr10": 133797422,
    }

    def score(signature):
        matches = 0
        total = 0
        for chrom, length in signature.items():
            val = refs.get(chrom) or refs.get(chrom.replace("chr", ""))
            if val:
                total += 1
                if val == length:
                    matches += 1
        return matches / total if total > 0 else 0

    score37 = score(GRCH37_SIG)
    score38 = score(GRCH38_SIG)

    if score37 > score38 and score37 > 0.6:
        return {
            "build": "GRCh37",
            "confidence": score37,
            "method": "dictionary_match"
        }

    if score38 > score37 and score38 > 0.6:
        return {
            "build": "GRCh38",
            "confidence": score38,
            "method": "dictionary_match"
        }

    return {
        "build": "unknown",
        "confidence": max(score37, score38),
        "method": "dictionary_ambiguous"
    }
🔬 Optional: SNP probing fallback (very robust)

This checks actual read alignment behavior.

import subprocess

def probe_position(bam_path, region):
    cmd = ["samtools", "mpileup", "-r", region, bam_path]
    result = subprocess.run(cmd, capture_output=True, text=True)
    return len(result.stdout.strip()) > 0


def detect_build_by_probe(bam_path):
    # Example known differing positions
    GRCH37_POS = "chr1:752566-752566"
    GRCH38_POS = "chr1:752721-752721"

    hit37 = probe_position(bam_path, GRCH37_POS)
    hit38 = probe_position(bam_path, GRCH38_POS)

    if hit37 and not hit38:
        return {"build": "GRCh37", "confidence": 0.9, "method": "probe"}
    if hit38 and not hit37:
        return {"build": "GRCh38", "confidence": 0.9, "method": "probe"}

    return {"build": "unknown", "confidence": 0.0, "method": "probe_failed"}
🧠 Master function (what you actually use)
def detect_genome_build(bam_path):
    # Step 1: fast path
    result = detect_build_from_header(bam_path)
    if result["build"] != "unknown":
        return result

    # Step 2: dictionary matching
    result = detect_build_from_full_dictionary(bam_path)
    if result["build"] != "unknown":
        return result

    # Step 3: probe fallback
    result = detect_build_by_probe(bam_path)
    return result
